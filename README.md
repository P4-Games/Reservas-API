## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

Para conectar tu base de datos modifica la variable que esta en src/app/db/conn.ts con tu URL de MongoDB Atlas

En /model tenes el schema que usa la base de datos.

Dentro de la Carpeta api podes ir creando carpetas y dentro de estas armas achivos route.ts esto enruta los llamados de tus distintas apis ejemplo http://localhost:3000/api/disponible
